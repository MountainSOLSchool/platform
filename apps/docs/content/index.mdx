# Mountain SOL Admin Guide

This is the internal reference for Mountain SOL **admins**. It covers every admin
tool and workflow in the enrollment portal — and includes a **Helping Families**
section so you can confidently answer questions families send your way (e.g.
*"Can I add a class to a registration I already paid for?"*).

It is **not** a parent-facing site. Families don't read workflow docs; they reach
out, and you use this guide to help them.

## How this guide is organized

- **[Getting Started](/getting-started/admin-access)** — how admin access works and a tour of the dashboard.
- **[Class Management](/class-management)** — create, edit, copy, and bundle classes; units, images, schedules.
- **[Semester Setup](/semester-setup)** — semesters, the enrollment-page info panel, banners, and confirmation-email content.
- **[Enrollment & Discounts](/enrollment-discounts)** — view, revoke, and refund enrollments; run discount codes.
- **[Students & Rosters](/students-rosters)** — student info, printouts, email exports, t-shirt counts.
- **[Medic Admin](/medic)** — the separate Medic program admin.
- **[Helping Families](/helping-families)** — answers to the questions families actually ask.
